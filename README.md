# SoniEffect Releases

Welcome to the official **SoniEffect release hub**! Here you can access the latest versions of SoniEffect, a modern **offline music player and audio player** optimized for smooth performance, playlist management, and advanced audio controls. Each release provides stable, tested builds ready for public or private distribution.

---

## About SoniEffect

SoniEffect is designed for listening to **local music files** with clarity, simplicity, and precision. It offers a clean interface, responsive playback, and practical audio tools for everyday use. The app focuses on:

- **Offline music playback** without unnecessary permissions
- Efficient **playlist management** and library navigation
- **Audio customization** including pitch, tempo, and metadata editing
- A lightweight, privacy-aware design that works with large music collections

---

## Release Highlights

Each release includes improvements, bug fixes, and new features. Key areas covered in SoniEffect releases include:

### High-Quality Offline Music Playback
- Stable background playback for uninterrupted listening
- Shuffle, loop, and repeat modes
- Fast song loading with minimal lag
- Optimized for large music libraries

### Local Music Library Management
- Organized library by **songs, albums, artists, playlists**
- Metadata display: title, artist, album, duration, artwork
- Quick library refresh and folder selection for full control

### Smart Default Playlists
- **Favorites**, **Recently Played**, **Most Played**
- Automatic updates based on listening behavior
- Fully manageable: clear, disable, or reset anytime

### Custom Playlist Manager
- Create, rename, or delete playlists
- Add/remove songs anytime
- Optional playlist description and cover image
- Lightweight and fast for large libraries

### Audio Control and Sound Tools
- Pitch and tempo adjustment
- Audio tag editor for metadata: title, artist, album, artwork, genre, year, lyrics
- Video to audio conversion for extracting audio from video files
- Safe file editing with no unnecessary access

### Video to Audio Converter
- Select video files manually for audio extraction
- Processed securely on server with download link
- Converted files are automatically removed after use
- Minimal permissions required

### Appearance and Visual Customization
- Light and dark mode
- Custom primary colors with RGB picker and presets
- Dynamic backgrounds based on album artwork
- Customizable song list layout

### Music Display and Sorting Options
- Album artwork, title, artist, duration
- Favorite, most played, and last played indicators
- Toggle display elements for minimal or detailed view

### App Updates and Version Checking
- Automatic or manual update notifications
- Continue using app if update skipped
- Manual update checks available in settings

### Privacy and Data Handling
- Audio permission only, no full storage access
- File access only for specific user actions
- Temporary storage for online processing, auto-deleted after use
